# ⚡ 洛克王国 · 对局战斗辅助工具

> 《洛克王国：世界》6v6 回合制辅助工具 —— 手动记录战局 + AI 辅助策略分析

> 🔗 立即体验：[https://rose0711.github.io/roco-battle/](https://rose0711.github.io/roco-battle/)

---

## 📖 项目介绍

本项目是一个面向《洛克王国世界》竞技玩家的**对战推演辅助工具**，用于**记录战局、复盘分析、辅助决策**。

**它不是自动战斗模拟器**，而是一个灵活的手动推演台 —— 你可以配置双方阵容，逐回合手动记录行动和伤害，再用内置 AI 策略分析功能获取下一回合的深度建议。

**核心特点：**

- 🎯 **纯前端** —— 浏览器直接打开即可运行，不需要服务器、不需要编译、不需要数据库
- 📚 **内置数据库** —— 收录了大量精灵、技能、特性、印记、状态数据，输入即自动补全
- 🧠 **AI 辅助** —— 接入 DeepSeek / 通义千问 API，可对当前战局进行深度推演分析
- 🗂️ **完整存档** —— 对局可存档/读档，阵容可命名保存/载入
- 🖼️ **完全本地化** —— 所有图片资源（精灵、技能、特性图标等）均为本地文件，无需联网加载

## ✨ 功能一览

### 阵容配置

- 配置我方 / 敌方各 **6 只精灵**
- **输入精灵名自动填充** —— 属性、特性、技能、克制/抵抗关系自动补全
- **首发精灵设置** —— 一键标记首发
- **敌方两种模式** —— 「已知全部精灵」或「未知阵容」（对战中动态发现）
- **阵容存档系统** —— 命名保存 / 载入阵容配置（JSON 格式）

### 对战推演

- **全面的状态追踪** —— 每只精灵的 HP、能量、属性、特性、克制/抵抗尽收眼底
- **血条系统** —— 4 点生命值，精灵阵亡扣血，❤️ / 🖤 可视化
- **丰富的行动选项** —— 释放技能 / 更换精灵 / 聚能回复 / 愿力冲击 / 首领化
- **属性克制自动显示** —— 克制 ×2、被克制 ×0.5、本系加成 +20%
- **世界印记系统** —— 正面 / 负面印记各一个，换宠不消失
- **战斗状态系统** —— 中毒、灼烧、寄生等异常状态，下场清空
- **敌军情报积累** —— 未知阵容模式下，可手动添加敌方新技能

### AI 策略分析

- **对战 AI** —— 自动读取当前战局，深度分析局势痛点、博弈预测、行动方案，给出下一回合最终指令
- **阵容 AI** —— 对配置的 6 只精灵进行战术分析，输出主力输出、辅助、联防、轮转思路
- **支持双 API** —— DeepSeek（推荐） / DashScope（阿里通义千问）
- **API Key 本地记忆** —— 可勾选「记住」，下次免输入

### 数据管理

- **对战存档** —— 完整保存所有回合记录，可在之后载入继续推演（浏览器 File System Access API）
- **CSV 数据引擎** —— 精灵、技能、印记等数据以 CSV 形式存储，易于维护和扩展
- **本地图片** —— 精灵头像、技能图标、特性图标等共计 1,400+ 张本地图片

## 🚀 运行方式

> 纯前端应用，无需构建、无需服务器，开箱即用。

**方式一：直接打开（最简单）**

```bash
# 克隆仓库
git clone https://github.com/rose0711/roco-battle.git
cd roco-battle

# Windows — 双击 index.html 或在终端运行
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

> ⚠️ **注意**：部分浏览器（如 Chrome）出于安全策略，禁止 `file://` 协议下的 `fetch` 请求。如果数据加载失败，请使用方式二。

**方式二：本地 HTTP 服务器（推荐）**

```bash
cd roco-battle

# Python 自带（Python 3）
python -m http.server 8080

# 或 Node 自带（需要安装 npm 包）
npx serve .

# 浏览器打开
http://localhost:8080
```

**方式三：VS Code Live Server（开发时推荐）**

1. 用 VS Code 打开 `roco-battle/` 目录
2. 安装扩展「Live Server」
3. 右键 `index.html` → 「Open with Live Server」

## 📋 使用方式

### 第一步：阵容配置

1. 打开页面进入**阵容配置视图**
2. 在「我方阵容」区域，依次输入 6 只精灵的名字
   - 输入时会自动弹出补全列表
   - 选择后，属性、特性、克制关系自动填充
   - 可选填写 HP 值和技能
3. 点击精灵卡片左上角的 **★** 设置首发精灵
4. 在「敌方阵容」区域，选择模式：
   - **🎯 已知全部精灵** —— 像我方一样完整配置敌方 6 只
   - **❓ 未知阵容** —— 只知道部分信息，对战中逐步发现
5. 点击 **💾 命名保存** 可将阵容存档，下次直接 **📂 载入存档**

> 💡 **AI 阵容分析**：填写完我方精灵后，在「阵容分析」区域填入 API Key，点击「AI 生成分析」，可自动获得阵容分工建议。

### 第二步：进入对战

点击底部绿色按钮 **⚔️ 进入对战**，进入对战视图。

界面分为左右两栏：
- **左侧（🟦 我方）** —— 出战精灵信息卡 + 后备精灵
- **右侧（🟥 敌方）** —— 出战精灵信息卡 + 后备精灵

每个信息卡显示：
- 精灵名字、属性标签
- 当前 HP / 最大 HP
- 当前能量 / 最大能量
- 特性（名称 + 描述）
- 克制 / 被克制 / 抵抗 / 弱抗关系
- 4 个技能（名称 + 类型 + 威力 + 耗能）

### 第三步：每回合操作

**回合流程：① 行动 → ② 结算 → ③ 下一回合**

#### ① 行动面板

在页面底部「行动面板」中，为双方选择本回合行动：

| 行动类型 | 说明 |
|---|---|
| **释放技能** | 选择四个技能之一，填写伤害/回复数值 |
| **更换精灵** | 从后备选择一只精灵上场 |
| **聚能** | 回复 5 点能量，本回合无法出招 |
| **愿力冲击** | 特殊行动选项 |
| **首领化** | 首领化不占回合，保留首领形态和新特性 |

> 换宠优先级 > 技能（换宠无视速度，优先执行）
> 技能出手顺序由先手值 + 速度共同决定

#### ② 结算面板

每回合行动后，在结算面板记录：
- 双方 HP 变化、能量变化
- 造成/受到的伤害值
- 阵亡精灵 → 自动扣除血量 → 选择替换上场
- 印记 / 状态的变化

#### ③ 生成下一回合

点击 **▶ 下一回合**，保存本回合数据，进入下一回合。

#### 🤖 AI 策略分析

在「AI 策略分析」区域（需填写 API Key）：
- 默认勾选 **「下一回合保存后自动分析」**
- 每回合保存后自动调用 AI 分析当前战局
- 分析结果包含：局势剖析、敌方博弈预测、我方行动方案、最终决策指令
- 也可手动点击 **🔄 重新分析** 随时调用

> 💡 **提示**：AI 分析用到了完整的战局快照数据（双方当前精灵状态、印记、已知技能等），分析结果会给出具体的技能名称和行动建议。

## 🔑 大模型 API 配置

> AI 分析功能需要调用大模型 API，两种接入方式任选其一。

### DeepSeek（⭐ 推荐）

DeepSeek 性价比高，对中文理解优秀，非常适合战术分析场景。

1. 访问 [DeepSeek 平台](https://platform.deepseek.com/) → 注册 / 登录
2. 进入左侧菜单 **「API Keys」**
3. 点击 **「创建 API Key」**，输入名称，复制生成的密钥（格式：`sk-xxx…xxxx`）
4. 在应用中选择 **DeepSeek** 作为大模型
5. 推荐模型：
   - `deepseek-v4-flash` —— 快速响应（适合日常分析）
   - `deepseek-v4-pro` —— 深度思考（适合复杂战局）
6. 建议勾选 **「记住」**，下次自动填充

> 💰 **费用参考**：DeepSeek 价格极低，日常使用几乎可以忽略。

### DashScope（阿里通义千问）

1. 访问 [DashScope 控制台](https://dashscope.aliyun.com/) → 注册 / 登录
2. 进入 **「API Key 管理」**
3. 点击 **「创建新 API Key」**，复制密钥
4. 在应用中选择 **DashScope** 作为大模型
5. 推荐模型：
   - `qwen-plus` —— 均衡之选
   - `qwen-max` —— 最强模型（精度更高）
6. 同样支持勾选「记住」

### ☁️ 试用模式

> 不想配置 API Key？可以直接使用内置的试用模式（每日 5 次），零配置体验 AI 分析功能。

1. 点击 **☁️ 试用模式** 按钮开启
2. 系统会自动使用内置代理调用 DeepSeek 进行 AI 分析
3. 无需填写任何 API Key，开箱即用
4. 每日 5 次试用额度，次日自动重置
5. 试用额度用完后，需要配置自己的 API Key 继续使用
